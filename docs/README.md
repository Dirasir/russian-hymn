# Russian Hymn Player - VS Code Extension

**Автор:** Байцуров Даниил Романович 501795
**Группа:** М3106

## Описание
Плагин для VS Code, который воспроизводит гимн России при нажатии сочетания клавиш.

## Функциональные требования
Воспроизведение аудиофайла гимна по команде
Автоматическое определение ОС пользователя
Для Windows: фоновое воспроизведение через PowerShell
Уведомления о статусе воспроизведения


## Использование
Нажмите `Ctrl+Shift+Z Ctrl+Shift+V` для воспроизведения гимна
